# Instrucciones para ejecutar la aplicación Vue

Para ejecutar la aplicación Vue, sigue los siguientes pasos:

1. Instala las dependencias utilizando npm:

    ```
    npm install
    ```

2. Luego, ejecuta el servidor de desarrollo con el siguiente comando:

    ```
    npm run dev
    ```

Esto iniciará el servidor de desarrollo y podrás acceder a la aplicación Vue en tu navegador.

# Instrucciones para hacer el build

Si deseas compilar la aplicación para su distribución, puedes hacerlo con el siguiente comando:

    ```
    npm run build
    ```

Esto creará una versión optimizada de tu aplicación Vue en el directorio "dist".

## Nota importante

Antes de ejecutar el comando `npm run build`, asegúrate de guardar los archivos actuales del directorio "dist" en un lugar seguro. Estos archivos son necesarios para convertir la aplicación en una Progressive Web App (PWA), y podrían sobrescribirse durante el proceso de compilación.
